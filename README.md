Student Affairs Office Website – IIT Patna
Overview

This project is a responsive frontend website developed for the Student Affairs Office (SAO), IIT Patna. It provides students with easy access to information related to student welfare, campus life, resources, support services, office initiatives, and contact information through a modern and user-friendly interface.

Features:
Home page with campus image carousel,
About Student Affairs Office section,
Key Responsibilities section with interactive cards,
Student Welfare and Support information,
Meet the Team section with profile cards,
Campus Life Gallery page,
Resources & Support section,
Frequently Asked Questions (FAQ),
Contact Information section,
CMS Login and Dashboard frontend prototype.

Technologies Used:
HTML,
CSS,
Bootstrap,
Vercel.

The website is optimized for desktop, tablet, and mobile devices.

CMS Information:

The CMS included in this project is a frontend prototype. Users can access it through the CMS Login button in the navigation bar. Authentication and backend functionality are not implemented in this version.

Live Deployment: https://student-affairs-office.vercel.app/
